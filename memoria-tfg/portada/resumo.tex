\begin{abstract}
  Este Trabajo Fin de Grado aborda el diseño, despliegue y operación de una plataforma cloud para la gestión de transportes sobre AWS. El sistema se construye como un backend contenerizado con persistencia en PostgreSQL, infraestructura definida mediante Terraform, automatización CI/CD y mecanismos de observabilidad y seguridad básica.
